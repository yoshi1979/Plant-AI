# Trusted source policy

The diagnosis validator should prefer these source classes:

- University extension services (`.edu`)
- Government agriculture resources (`.gov`)
- Royal Horticultural Society (`rhs.org.uk`)
- Missouri Botanical Garden (`missouribotanicalgarden.org`)
- American Phytopathological Society (`apsnet.org`)
- Additional vetted plant-care databases only if explicitly approved

Rules:
- Do not present weak blogs or forum posts as authoritative evidence.
- If strong sources are unavailable, lower confidence.
- If sources conflict, mark validation as conflicting and ask follow-up questions.
- Avoid extreme treatment recommendations without strong evidence.
